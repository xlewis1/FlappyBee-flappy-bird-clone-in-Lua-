# 🐝 Flappy Bee

A polished, "one-more-try" arcade game inspired by the classic *Flappy Bird*. Navigate through flower stems, manage your momentum, and try to secure the ultimate high score!

## 🎮 Play Live
The game is compiled into WASM and hosted directly on GitHub Pages!
👉 **[Play Flappy Bee Here](https://xlewis1.github.io/FlappyBee-flappy-bird-clone-in-Lua-/)**

---

## 🛠️ Built With
* **Language:** Lua
* **Framework:** LÖVE (Love2D)
* **Web Deployment:** love.js (Embedded via WebAssembly / WASM)
